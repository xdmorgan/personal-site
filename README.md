- Hosted on Netlify
- Styles generated by [Skeletor](https://www.npmjs.com/package/@skeletor/css)

## Scripts

1. `$ yarn`: Install deps
1. `$ yarn test`: Verify installation
1. `$ yarn start` or `$ yarn develop`: Run dev server
1. `$ yarn preview` or `yarn build && yarn serve`: Build and serve
